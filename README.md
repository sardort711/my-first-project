# PDP University Dashboard & Student Portal 🎓

Welcome to the premium, interactive **PDP University Student Dashboard**. This frontend project was structurally redesigned from the ground up to feature modern glassmorphism UI/UX aesthetics, 3D interactive elements, and highly dynamic student system tracking mechanisms.

## 🚀 Key Features

*   **Global Live Search Engine:** Instantly index and filter visually matched data (Academic Subjects, Mentors, and Class Schedules) straight from the top-header search bar in real time.
*   **Immersive 3D ID Card:** Features a precise 3:2 horizontal aspect ratio `preserve-3d` rotating student ID Badge on the profile page.
*   **Mathematical Attendance Analytics:** Custom, logic-driven JavaScript tooltips hover accurately over dynamic subject progress bars, explicitly detailing the exact ratios of *Present*, *Absent*, *Excused*, and *Compensated* metrics for every unique class.
*   **Automated Helpdesk Chatbot:** An embedded support module featuring functional JS text-capture and automated system responses seamlessly locked to the bottom-right viewport.
*   **Interactive Payment Gateway:** A dedicated Financial module that overlays a high-fidelity glassmorphic checkout UI, highlighting precise tuition balances with active gateways for **Payme**, **Click Up**, and **Bank Cards**.
*   **Dynamic Theme Rendering:** A beautiful, responsive light-mode GUI utilizing custom SVG icons, vibrant gradient arrays, box-shadow depth mapping, and fluid `:hover` transition states.

## 📂 Project Structure

This is a **pure frontend Vanilla architecture**. It deliberately relies on absolutely zero third-party component libraries (No Bootstrap, no Tailwind, no React)—everything is custom-engineered using native DOM logic and CSS3 properties.

*   `dashboard.html` — The primary Hub. Houses the global Search Bar, Stats Grid, embedded Native Weekly Timetable, and Academic Performance trackers.
*   `profile.html` — The standalone Student Portal window containing the 3D rotating Student ID, detailed Personal Information matrices, and Financial/Payment modules.
*   **Assets:**
    *   `student.JPG` *(Profile Avatar)*
    *   `front.png` & `back.png` *(3D ID Card Graphics)*
    *   `mentor.png` *(Mentorship Graphics)*
    *   `payme.png` & `click.png` *(Payment Gateway Icons)*

## ⚙️ How to Run

1.  Make sure all graphic files (`.png`, `.JPG`) are located in the exact same directory as your HTML files.
2.  Simply double-click `dashboard.html` to launch the portal in any modern browser (Google Chrome or Microsoft Edge recommended for optimal transition and 3D rendering compliance).
